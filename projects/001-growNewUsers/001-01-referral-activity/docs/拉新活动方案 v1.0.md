# 拉新活动方案 v1.0

---

## 一、活动目的

让老玩家拉新玩家进来玩，给金币奖励。拓展新增渠道。

---

## 二、活动流程（7步走）

| 步骤 | 环节 | 说明 |
|------|------|------|
| 第一步 | 预热 | 给团长和活跃玩家发活动通知，告诉他们怎么参与（这块是否需要拉人成为团长有待讨论） |
| 第二步 | 分享 | 邀请人把专属邀请码或链接分享给朋友，配上我们给的话术模板 |
| 第三步 | 注册填码 | 新玩家下载游戏后，在注册页面填写邀请人的邀请码 |
| 第四步 | 绑定手机号 | 新玩家必须绑定手机号才能完成注册（这是为了防止一个人开小号刷奖励） |
| 第五步 | 进游戏激活 | 新玩家真正进入游戏玩起来 |
| 第六步 | 发基础奖励 | 系统自动给邀请人发拉新奖励金币 |
| 第七步 | 首充额外奖励 | 如果被邀请人完成了首次充值，邀请人可以再获得一笔返点金币 |

---

## 三、在哪里填邀请码

1. **注册页面（主入口）**
   - 必填项，填写后才能注册成功

2. **分享链接自动识别（辅助）**
   - 邀请人分享专属链接，别人点击后直接跳到下载页，注册时自动填好邀请码

> 现状：邀请码功能和分享链接目前都**[待开发]**，需要技术评估实现方式

---

## 四、奖励规则

### 4.1 基础拉新奖励

| 角色 | 奖励内容 | 发放条件 |
|------|----------|----------|
| 邀请人 | 30 金币/人 | 被邀请人完成注册、绑定手机号并产生游戏行为数据 |
| 被邀请人 | 50 金币 | 完成注册、绑定手机号并产生游戏行为数据，发放到第一个产生游戏行为数据的服务器 |

> 发放时机：以产生游戏行为数据为准，确保是真实活跃用户

### 4.2 首充返点奖励

| 项目 | 内容 |
|------|------|
| 触发条件 | 被邀请人完成首次充值 |
| 奖励比例 | 首充金额的 20% 转化为金币 |
| 发放对象 | 邀请人 |
| 金额限制 | 不设上限 |
| 时效限制 | 不设时间限制，任何时候首充均可触发 |

### 4.3 金币发放方式（邀请人）

| 项目 | 说明 |
|------|------|
| 发放位置 | 不直接发放到游戏账户，进入专属金币池 |
| 领取方式 | 邀请人在专属管理界面自主设置领取到哪个服务器 |
| 查看内容 | 专属邀请码和邀请链接、邀请名单、激活状态、首充状态及返点金额、累计奖励总额 |

## 五、防刷奖励机制（防薅羊毛）

### 5.1 核心手段：手机号实名制

| 项目 | 说明 |
|------|------|
| 规则 | 越南一人可以有3个手机号，法律上需要实名认证，但是实际监管可能没那么严格，所以将手机号（成本 ¥50 ）以及灰产手机号成本（估算 ¥25 ）作为金币发放的测算依据使得刷金币成本高于收益 |
| 邀请上限 | 活动初期可设置每人最多邀50人，观察数据后再放开 |
