# 团长运营手册 (Guild Leader Operations Manual)

## 核心使命

团长 (Guild Leader) 是战斗指挥与团队灵魂。

原则：
- 无敌人的团队是一盘散沙
- 无冲突的游戏无法长久

任务：制造外部矛盾，激发团员"胜负欲"与"保护欲"，提高活跃度，促进充值。

---

## 第一阶段：日常冲突制造

目标：非决战时间保持"火药味"，让团员感觉"有人在欺负我们"，产生复仇心理。

### 1.1 骚扰战术

#### 领地骚扰

| 项目 | 说明 |
|------|------|
| 频率 | 每天不定时 |
| 行动 | 组织小分队去其他团队领地"溜达" |
| 偷袭 | 利用敌方领地功能未完全开启的空窗期，针对性击杀敌方在养恐龙 |

#### 运营话术

> "兄弟们，对面那只霸王龙我看它不顺眼很久了，今晚把它宰了！"

### 1.2 心理战术

#### 喷漆嘲讽

| 项目 | 说明 |
|------|------|
| 功能 | 游戏喷漆功能 (Spray Paint) |
| 操作 | 潜入敌方领地核心区域，喷涂挑衅图案或文字 |
| 效果 | "骑在头上拉屎"行为，最能激怒对方团长和成员 |

#### 公屏喊话

| 项目 | 说明 |
|------|------|
| 频道 | 世界频道/公屏 |
| 尺度 | 可嘲讽对方"怂"、"不敢打"、"菜" |
| 严禁 | 人身攻击或违禁词 |

#### 话术示例

> "XX公会的龙怎么这么脆？轻轻一碰就没了，团长没给喂饭吗？"

## 第二阶段：宣战流程

目标：摩擦积累到一定程度后，将冲突正式化，通过仪式感极强的"宣战"动员全员。

### 2.1 宣战发起

#### 触发条件

团长决定宣战。

#### 操作流程

| 步骤 | 操作 |
|------|------|
| 1 | 联系游戏客服 (GM) 申请宣战 |
| 2 | 提交宣战词（热血、霸气，客服将全服公告） |
| 3 | 设定奖惩方案（胜方奖励明显更丰厚，刺激欲望） |
| 4 | 赌约：1000 金币，看对方接不接 |

### 2.2 对方接战

| 项目 | 说明 |
|------|------|
| 决策时间 | 敌方团长 12 小时决策期 |
| 公告 | 确定后客服发全服公告 |
| 决战 | 次日进行 |

#### 规则限制

| 限制 | 说明 |
|------|------|
| 每日宣战次数 | 每个团队每天只能发起 1 次 |
| 最大拒绝次数 | 每个团队最多只能拒绝 2 次（"躲得过初一躲不过十五"，逼迫应战） |

---

## 第三阶段：决战时刻

### 3.1 战斗规则

| 项目 | 说明 |
|------|------|
| 时间 | 21:00 - 24:00 |
| 地点 | 管理山 (Admin Mountain) |
| 机制 | 团队 PK |
| 胜利条件 | 坚持到底 |
| 失败条件 | 认输 |

### 3.2 场外干预机制

| 项目 | 说明 |
|------|------|
| 封闭式决斗 | 仅限宣战双方进入战斗区域 |
| 神罚机制 | 第三方团队干扰时，客服直接介入，操作"天雷"劈死干扰者恐龙 |

### 3.3 宣战词规范

原则：动员令，拒绝干巴巴。

#### 差示例

> "我们要打XX公会了。"

#### 好示例

> "XX公会多次偷袭我方幼龙，此仇不报非君子！今晚9点管理山，赌上公会的荣耀，不死不休！全员集结，杀！"

## 第四阶段：日常小游戏

### 4.1 竞速类：三角龙拉力赛

| 项目 | 说明 |
|------|------|
| 路线 | 大地图规划（如河流一段） |
| 参赛 | 所有参赛者必须用三角龙 (Triceratops) |
| 规则 | GM 起点发令，同时出发；途中可互相攻击；GM 终点等待或沿途裁判 |
| 奖励 | 前三名获得奖励 |

### 4.2 趣味挑战类：进击的雷龙

| 项目 | 说明 |
|------|------|
| 触发 | GM 公告，特定地点（如工厂）刷出变异雷龙 |
| 变异属性 | 体型放大 2 倍、防御力和血量极高 |
| 规则 | 全服玩家或特定公会成员前往围剿；GM 准备过程公告 |
| 奖励 | 最后一击的玩家/队伍获得奖励 |

### 4.3 趣味挑战类：追杀踢脚鸡

| 项目 | 说明 |
|------|------|
| 触发 | GM 公告，特定地点刷出特殊踢脚鸡 |
| 外观 | 颜色鲜艳皮肤 |
| 属性 | 体积调小（难选中）、移速调高（难追）、防御力适中 |
| 行为 | 工厂随机乱跑 |
| 规则 | 全服玩家或特定公会成员前往围剿 |
| 奖励 | 最后一击的玩家/队伍获得奖励 |

### 4.4 趣味休闲类：全服躲猫猫

| 项目 | 说明 |
|------|------|
| GM 操作 | 操控极小恐龙（如幼年犹他龙 Juvenile Utahraptor）在地图某区域闲逛 |
| 提示 | GM 只在大频道给出模糊提示（如："我看到了瀑布，旁边有红色的花"） |
| 胜利条件 | 第一个找到并站在 GM 面前跳舞/做动作的玩家 |
| 奖励 | 获胜玩家获得奖励 |

| 参数名 | 数值 | 说明 |

| 宣战响应时间 | 12 小时 | 敌方决定期 |
| 每日宣战次数 | 1 次 | 发起方限制 |
| 最大拒绝次数 | 2 次 | 接收方限制 |
| 团战时段 | 21:00-24:00 | 管理山 PK |
| 宣战赌约 | 1000 金币 | 基础赌注 |
| 小保护时段 | 00:00-08:00 | 领地保护 |
| 大保护次数 | 每月 2 次 | 公会可开启 |
