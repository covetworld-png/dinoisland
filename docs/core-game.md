# 核心游戏设定 (Core Game Settings)

## 1. 世界观 (World Setting)

人类在恐龙岛上建立秘密实验室复活恐龙。未知灾难降临，人类离奇消失，留下废弃建筑与成群恐龙军团。现为恐龙世界。

## 2. 服务器与大厅

| 项目 | 说明 |
|------|------|
| 数据互通 | 不同服务器数据不互通 |
| 新手建议 | 选择人数较少服务器，减少卡顿与生存压力 |

### 2.1 游戏流程

1. 选择服务器进入大厅
2. 创建恐龙：选择恐龙从幼龙 (Juvenile) 开始冒险
3. 仓库管理：取出已存档恐龙或配置喷漆等道具
4. 商城：购买恐龙、礼包及功能卡

### 2.2 注意事项

进入游戏世界后无法返回仓库。

## 3. 游戏界面 (UI) 解析

### 3.1 状态栏（右下角）

需时刻关注恐龙生理指标：

| 指标 | 说明 |
|------|------|
| 生命值 (HP) | 归零死亡 |
| 饱食度 (Satiety) | 影响体力恢复 |
| 水分 (Hydration) | 影响移动速度 |
| 体力 (Stamina) | 快跑/特殊动作消耗 |
| 进化值 (Evolution Value) | 满 100% 可进化 |

## 4. 操作指南 (Control Guide)

### 4.1 基础移动

| 按键 | 功能 |
|------|------|
| W/A/S/D | 移动方向 |
| Z | 切换步行/小跑 |
| Shift + 移动 | 快跑（消耗体力） |
| Ctrl | 下蹲；翼龙为降落 |
| H | 趴下休息（加快体力恢复） |
| Space | 跳跃；翼龙为飞行上升 |

#### 蓄力技巧

下蹲一段时间后进入蓄力状态，此时按住 Shift 移动可触发加速突进。

### 4.2 生存互动

| 按键 | 功能 |
|------|------|
| E / R | 进食或喝水（需靠近食物/水源）；补充生命、饱食、水分，增加进化值 |
| F | 叼肉（需靠近尸体） |
| Q（按住） | 嗅探模式 (Scent Mode)：高亮显示附近食物和水源 |
| 鼠标左键 | 普通攻击 |
| 鼠标右键 | 特殊能力（如犹他龙滑行）或特殊攻击 |
| Shift + 左键 | 部分恐龙（如雷龙）特殊攻击 |

### 4.3 社交与表情

| 按键 | 功能 |
|------|------|
| Enter | 打开/发送聊天 |
| [ / ] | 切换世界/公会/系统频道（需在打开聊天框前） |
| 1-4 | 不同吼叫声 |
| 5 | 跳舞 |
| 6 | 装死 |
| 7 | 下跪（下跪后按 8 可磕头） |
| T | 喷漆（需先在仓库配置喷漆卡，游戏中按 Y 选定图案，对准地面按 T） |

### 4.4 系统功能键

| 按键 | 功能 |
|------|------|
| Esc | 系统菜单（设置、进化、保存退出、每日任务） |
| Tab | 信息面板（属性、皮肤、商城、公会、玩家列表/ID） |
| M | 打开地图 |
| X | 开启/关闭夜视仪 (Night Vision) |
| F10 | 隐藏 UI（截图模式） |
| F11 | 切换全屏/窗口模式 |

## 5. 核心玩法：进化与战斗

### 5.1 进化系统 (Evolution System)

| 项目 | 说明 |
|------|------|
| 条件 | 非成年恐龙 |
| 方法 | 喝水、进食积累进化值至 100% |
| 操作 | 确保各项状态（血量、饱食等）为满值 → Esc → 点击"进化" |
| 捷径 | 使用商城进化卡直接充满进化值 |

### 5.2 战斗技巧

| 机制 | 说明 |
|------|------|
| 受伤 | 被咬伤掉血，概率骨折 |
| 走位 | 转圈走位躲避攻击，攻击敌方侧后方 |
| 自杀指令 | Enter 输入 `/zs` 自杀回大厅 |

### 5.3 巢穴系统 (Nest System)

| 操作 | 说明 |
|------|------|
| 创建 | 成年母龙平地按 B |
| 管理 | 巢穴上按 H 趴下，再按 B 打开面板 |
| 拉取玩家 | 可拉取其他玩家（被拉取玩家原恐龙消失，谨慎操作） |
| 传送 | 可作为变相传送手段 |

## 6. 地图与传送

### 6.1 关键地点

| 地点 | 说明 |
|------|------|
| 工厂 (Factory) | 核心场景 |
| 地下实验室 (Underground Lab) | 核心场景 |
| 管理山 (Admin Mountain) | 主要 PVP 与社交区域 |

### 6.2 传送限制

只有恐龙状态全满时才可使用传送功能。

### 6.3 传送方式（按 M 键）

| 类型 | 消耗 | 说明 |
|------|------|------|
| 公共传送点 | 100 积分/次 | - |
| 公会传送点 | 100 积分/次 | 需加入公会 |
| 个人传送点 | 1 次传送次数 | 需拥有传送卡；设置需消耗定点卡 |

### 6.4 坐标存取（高风险操作）

保存恐龙后可在仓库提取坐标给新龙使用。

警告：使用此功能会"顶掉"旧恐龙，被顶掉的恐龙将永久消失。

## 7. 商城与充值系统

### 7.1 货币体系

| 货币 | 获取方式 | 用途 |
|------|---------|------|
| 积分 (Points) | 免费获取 | 基础消费 |
| 兽币 (Beast Coins) | 充值获取 | 高级消费 |

### 7.2 积分获取途径

- 完成每日任务（Esc 查看右侧）
- 击杀其他恐龙

### 7.3 兽币充值流程

1. F11 切换窗口模式（防止浏览器拦截）
2. 商城页面金币旁 + 号
3. 跳转网页登录，选择正确服务器
4. 填入用户 ID（Tab 玩家列表中查看）

## 8. 公会与军团战 (PVP 核心)

### 8.1 领地保护机制

| 保护类型 | 时间/次数 | 效果 |
|---------|----------|------|
| 小保护 | 每晚 00:00 - 08:00 | 敌方高阶恐龙（巨人龙、超霸、雷龙、机霸等）无法在领地内造成伤害 |
| 大保护 | 每月 2 次 | 期间任何恐龙都无法在领地内造成伤害 |

### 8.2 宣战系统

| 项目 | 说明 |
|------|------|
| 流程 | 会长发起宣战 → 敌方 12 小时内决定 → 接受则当晚 21:00-24:00 管理山团战 |
| 限制 1 | 每个团队每天只能发起 1 次宣战 |
| 限制 2 | 每个团队最多只能拒绝 2 次宣战 |
| 奖励 | 胜利方获得丰厚奖励 |
