# 国际化 (i18n) 工作流程手册

## 文件说明

| 文件 | 用途 | 谁操作 |
|------|------|--------|
| `i18n.json` | **主配置** - 所有翻译的权威来源 | 开发 |
| `index.html` | **页面文件** - 包含内嵌翻译 | 开发 |
| `i18n-for-translation.md` | **翻译文档** - 发送给翻译确认 | 翻译人员查看 |
| `i18n-tool.js` | **工具脚本** - 生成文档/嵌入翻译 | 开发执行 |

## 详细操作步骤

### Step 1: 开发更新代码

当新增或修改文本时：

1. 在 HTML 中使用 `data-i18n` 属性：
```html
<button data-i18n="common.登录">登录</button>
```

2. 在 `i18n.json` 中添加翻译：
```json
{
  "common": {
    "登录": {
      "vi": "Đăng nhập",
      "cn": "登录"
    }
  }
}
```

---

### Step 2: 生成翻译文档

```bash
cd src/campaign/pages/streamer-dropin
node i18n-tool.js export
```

输出：
```
✅ 已生成: i18n-for-translation.md (201 条翻译)
```

---

### Step 3: 发送给翻译

将 `i18n-for-translation.md` 发送给翻译人员。

**附言模板：**
> 请查看附件中的翻译文档。
> - 如需修改，请直接在文档中标注（加粗、下划线或批注）
> - 返回后我们会手动同步到系统

---

### Step 4: 翻译标注修改

翻译人员打开 MD 文件，找到需要修改的行，标注建议：

```markdown
| Key | 中文 (cn) | 越南语 (vi) | 状态 | 备注 |
|-----|-----------|-------------|------|------|
| common.登录 | **立即登录** | Đăng nhập | 建议修改 | 更紧迫 |
| common.价格 | 价格 | ~~Giá~~ **Giá cả** | 建议修改 | 更口语化 |
```

---

### Step 5: 开发对比更新

收到翻译返回的文档后：

#### 方法 1: VS Code 对比（推荐）

1. 保存原文为 `old.md`
2. 保存修改版为 `new.md`
3. 选中两个文件 → 右键 → "Compare Selected"

#### 方法 2: 在线工具

- 访问 https://www.diffchecker.com/
- 粘贴原文和修改版

#### 方法 3: 命令行

```bash
diff old.md new.md
```

#### 确认后更新 i18n.json

根据对比结果，手动修改 `i18n.json` 中的对应条目。

---

### Step 6: 嵌入到页面

```bash
node i18n-tool.js embed
```

输出：
```
✅ 已嵌入: index.html (201 条翻译)
```

## 常见问题

### Q: 翻译可以直接修改 i18n.json 吗？
A: 不建议。JSON 格式严格，容易因缺少引号/逗号导致解析错误。

### Q: 如何确认翻译修改了哪些内容？
A: 使用对比工具（推荐 VS Code），所有差异会高亮显示。

### Q: 可以部分应用翻译的修改吗？
A: 可以。对比后选择性修改 i18n.json，不需要全部采纳。

### Q: 嵌入 HTML 后需要测试吗？
A: 建议测试：
1. 打开页面
2. 点击语言切换按钮
3. 确认文本正确切换
