## 价格结构

### 亲卫勇士礼包

| 价格类型 | 金币 | VND (按钮显示) | 折扣 |
|---------|------|----------------|------|
| **原价** (划线) | 5,888 | - | - |
| **胜者价** (高亮) | 2,722 | 4,899,600đ | **-54%** |
| **败者价** (高亮) | 3,110 | 5,598,000đ | **-47%** |

**折扣计算:**
```
胜者折扣 = (5888 - 2722) / 5888 = 3166 / 5888 = 53.77% ≈ 54%
败者折扣 = (5888 - 3110) / 5888 = 2778 / 5888 = 47.18% ≈ 47%
```

## 配置代码

### config.json

```json
{
  "pricing": {
    "exchangeRate": 1800,
    "warrior": {
      "displayOriginal": 5888,
      "finalPriceWinner": 2722,
      "finalPriceLoser": 3110,
      "discountPercentWinner": 54,  // (5888-2722)/5888
      "discountPercentLoser": 47    // (5888-3110)/5888
    },
    "overlord": {
      "displayOriginal": 18888,
      "salePrice": 10888,
      "discountPercent": 42          // (18888-10888)/18888
    }
  }
}
```

### index.html (JavaScript)

```javascript
const SKIN_PRICES = {
    warrior: {
        get finalWinner() { return 2722; },
        get finalLoser() { return 3110; },
        discountPercentWinner: 54,  // 约54%
        discountPercentLoser: 47    // 约47%
    },
    overlord: {
        displayOriginal: 18888,
        salePrice: 10888,
        discountPercent: 42          // 约42%
    }
};
```
