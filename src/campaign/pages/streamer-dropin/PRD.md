## 3. 核心功能模块

### 3.1 阵营系统

| 服务器 | 主播A | 主播B | 对抗名称 |
|--------|-------|-------|---------|
| Q服 | heni | ricon | Quân Đoàn Chiến |
| K服 | Sữa Chua | MiuMiu | Khủng Long Chi Chiến |

**用户流程：**
1. 访问页面 → 查看 VS 卡片
2. 登录游戏账号
3. 进入游戏创建角色（激活状态）
4. 选择阵营加入
5. 加入对应 Discord 接收战术指挥

**限制规则：**
- 每账号每服限加入一个阵营
- 报名截止后不可更改
- 仅限已激活玩家加入

### 3.2 皮肤商城

| 皮肤类型 | 名称 | 原价(VND) | 折扣逻辑 |
|----------|------|----------|---------|
| 亲卫勇士 | 主播限定皮肤(4款) | 6,998,400 | 获胜阵营50% / 落败阵营10% |
| 史前霸主 | 超稀有礼包 | 33,998,400 | 固定34%折扣 |

**状态流转：**
```
comingSoon(即将上线) → revealed(已公开) → onSale(开售) → eventEnded(结束) → skinDelivered(已发放)
```

### 3.3 多语言支持

| 语言 | 代码 | 覆盖范围 |
|------|------|---------|
| 越南语 | vi | 默认语言，完整翻译 |
| 中文 | cn | 备用语言 |

**实现方式：**
- 翻译文件：`i18n.json` / `locales/*.ts`
- 切换方式：页面右上角语言按钮
- 存储：用户选择存入 localStorage

## 4. 配置管理

### 4.1 可配置项

| 配置类别 | 具体项 | 配置方式 |
|----------|--------|---------|
| **时间节点** | 报名截止、对抗开始、皮肤公布、开售结束 | `config.json` |
| **阵营结果** | 获胜方、折扣比例 | `config.json` |
| **皮肤状态** | 当前显示阶段 | `config.json` |
| **定价** | 金币价格、汇率、折扣 | `config.json` |
| **链接** | Discord、下载源 | `config.json` |

### 4.2 配置加载机制

```javascript
// CDN 配置中心
const config = await fetch('https://cdn.monster-lair.vn/campaign/config.json');

// 本地缓存 5 分钟
localStorage.setItem('campaign_config', JSON.stringify(config));
```

## 5. 数据埋点

### 5.1 转化漏斗

| 层级 | 事件 | 关键指标 |
|------|------|---------|
| 访问 | `page_view` | UV、来源渠道、设备类型 |
| 登录 | `login_success` | 登录转化率 |
| 激活 | `character_check` | 游戏激活率 |
| 加入阵营 | `faction_join_success` | 阵营选择分布 |
| 购买 | `purchase_complete` | 付费率、ARPU |

### 5.2 事件清单

详见 [README.md](./README.md) 数据埋点章节

### 5.3 上报机制

- **批量上报**：队列满 10 条或 30 秒自动上报
- **失败重试**：最多 3 次，存入本地队列
- **匿名追踪**：未登录用户生成匿名 ID
