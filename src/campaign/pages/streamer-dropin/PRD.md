# 主播空降恐龙岛活动 PRD

## 1. 活动概述

### 1.1 活动背景
4位头部主播空降《恐龙岛》游戏，带领玩家进行阵营对抗，通过主播影响力拉动新增、激活付费。

### 1.2 活动目标

| 目标 | 指标 | 目标值 |
|------|------|--------|
| **拉新** | 活动页注册用户 | 5,000+ |
| **活跃** | 活动页 UV | 10,000+ |
| **付费** | 皮肤购买转化率 | >5% |
| **社交** | Discord 加入人数 | 3,000+ |

### 1.3 活动时间

| 阶段 | 时间 | 说明 |
|------|------|------|
| 预热期 | 3/25 - 3/31 | 阵营报名、皮肤悬念 |
| 爆发期 | 4/1 20:00 | 主播对抗直播 |
| 销售期 | 4/2 - 4/17 | 皮肤开售、结果折扣 |

## 2. 业务逻辑

### 2.1 阵营对抗规则

```
Quân Đoàn Chiến (Q服)
├── heni 阵营
└── ricon 阵营

Khủng Long Chi Chiến (K服)  
├── Sữa Chua 阵营
└── MiuMiu 阵营
```

**胜负判定** (V2 更新)：
- 爆发期直播结束后，根据各阵营在线人数/击杀数决定胜负
- 获胜方阵营皮肤折扣 **-54%** (原价 5888 金币 → 2722 金币)
- 落败方阵营皮肤折扣 **-47%** (原价 5888 金币 → 3110 金币)

### 2.2 皮肤销售规则 (V2 更新)

| 皮肤类型 | 原价 | 最终价 | 折扣 | VND (按钮显示) |
|----------|------|--------|------|----------------|
| 亲卫勇士-胜者 | 5,888 金币 | 2,722 金币 | **-54%** | 4,899,600đ |
| 亲卫勇士-败者 | 5,888 金币 | 3,110 金币 | **-47%** | 5,598,000đ |
| 史前霸主 | 18,888 金币 | 10,888 金币 | **-42%** | 19,598,400đ |

**价格显示规则**：
- 价格区域**仅显示金币**，原价划线、最终价高亮
- 越南盾价格仅在**购买按钮**上显示

**购买资格**：仅限已加入阵营玩家

## 6. 数据评估模型

### 6.1 核心指标定义

| 指标 | 计算公式 | 采集方式 |
|------|----------|----------|
| **新增用户** | 活动页注册且创建角色 | `login_success` + `character_check` |
| **激活率** | 创建角色数 / 注册数 | `faction_join_success` / `login_success` |
| **阵营转化率** | 加入阵营数 / 激活数 | `faction_join_success` / UV |
| **购买转化率** | 购买数 / 加入阵营数 | `purchase_complete` / `faction_join_success` |
| **ARPU** | 总销售额 / 付费用户数 | 订单金额求和 / 付费人数 |

### 6.2 归因模型

```
用户访问 URL 格式：
https://monster-lair.vn/campaign/streamer-dropin/?source=facebook&utm_campaign=streamer_dropin_mar&utm_content=heni_video

归因字段：
- source: 流量来源（facebook/google/youtube/direct）
- utm_campaign: 活动标识
- utm_content: 素材/主播标识（heni_video/ricon_live等）

数据聚合维度：
1. 渠道维度：各 source 的转化率对比
2. 主播维度：各 streamer 的拉新效果对比
3. 素材维度：各 content 的点击率对比
```

### 6.3 漏斗分析

```
100%  页面访问 (page_view)
  │
  ├─ 30% ──► 登录 (login_success) ──────► 流失分析：为何70%未登录？
  │
  ├─ 20% ──► 激活 (has_character=true) ──► 流失分析：下载到激活的转化
  │
  ├─ 15% ──► 加入阵营 (faction_join) ────► 流失分析：阵营吸引力
  │
  └─ 5%  ──► 购买 (purchase_complete) ───► 流失分析：价格敏感度
```

### 6.4 AB 测试建议

| 测试项 | 变量A | 变量B | 评估指标 |
|--------|-------|-------|----------|
| 倒计时紧迫感 | "还剩14天" | "仅剩14天" | 登录转化率 |
| CTA按钮文案 | "加入阵营" | "立即参战" | 阵营转化 |
| 皮肤预览 | 静态图 | 动态视频 | 购买转化 |
